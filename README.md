Guessing Game

This is a Python program for a fun guessing game where users try to guess a random number between 1 and 100. 
Players have up to 7 attempts, and the program provides feedback on whether the guess is "Too High" or "Too Low." 

Features
- Friendly prompts for user input.
- Handles invalid inputs gracefully.
- Tracks how long it takes the user to complete the game.
- Encourages replayability!

How to Play
1. Download the code from this repository.
2. Run the Python file using any Python interpreter.
3. Follow the prompts to guess the number!

Enjoy the game, and feel free to share your thoughts!
